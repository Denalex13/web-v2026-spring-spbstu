Задача 5 Реализовать механику компонента “подробнее”
Создать список книг, который будет содержать обложку книги, название.
Будет параграф с описанием книги. Он должен содержать достаточно объемный
текст, помещающийся на половину экрана (5-6 предложений). Необходимо
скрывать описание до 1 строки и показывать кнопку “Развернуть”.
Функциональность:
- показывает весь текст, при нажатии кнопки “Развернуть”;
- скрывает часть текста, при нажатии кнопки “Свернуть”;
- текст должен плавно сворачиваться и разворачиваться.
