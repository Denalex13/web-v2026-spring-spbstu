4.1. Создать класс channel с полями:
• id — номер (число)
• name — название канала (строка)
• subscribers — массив имён подписчиков (строки)
Реализовать методы:
• addSubscriber(name) — добавить подписчика
• removeSubscriber(name) — удалить подписчика
• геттер subscriberCount — количество подписчиков
4.2. Дан массив каналов:
• Группировать каналы по количеству подписчиков
• Получить уникальный список всех подписчиков
• Вернуть каналы с заданным подписчиком
• Группировать подписчиков по каналам (подписчик → массив каналов)
• Получить каналы с максимальным количеством подписчиков
4.3. Реализовать пользовательский интерфейс:
• Карточки каналов: название, список подписчиков, количество
подписчиков
• Кнопки: "Добавить подписчика", “Добавить подписчика”, “Добавить
канал”, "Удалить канал"
• Форма создания канала. Форма добавления подписчика
• При любых изменениях автоматически обновлять пользовательский
интерфейс
• Все операции добавления/удаления выполнять асинхронно через Promise
+ setTimeout()
• Использовать local storage для сохранения текущего списка студентов при
закрытии вкладки.
